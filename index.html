Introduction

The Covid-19 lockdownds during 2020 had a large impact on the US labor market. Using MySQL for analysis and Javascript for data modeling, I queried as many datasets as I could find that reflected changes due to covid. The data shows us how unenployment rose, which industries were affected the most, and if any companies on found success during this time.
